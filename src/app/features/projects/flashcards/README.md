# Flashcards

Flashcards is a LocalStorage productivity mini project for studying small decks of question and answer cards.

## Features

- Deck filtering
- Answer reveal
- Correct and incorrect review actions
- Progress and mastered-card counts
- Shuffle
- Add, edit, and delete cards
- Demo reset
- LocalStorage persistence

## Angular Coverage

- Standalone component
- Template-driven forms
- Signals and computed state
- LocalStorage persistence
- Validation state for required question and answer fields

## Limitations

- The project is intentionally local-only and does not sync cards across devices.
- Decks are inferred from card data instead of managed as separate records.

## Future Work

- Add spaced-repetition scheduling.
- Add import/export for decks.
- Add keyboard shortcuts for reveal, correct, and incorrect actions.
