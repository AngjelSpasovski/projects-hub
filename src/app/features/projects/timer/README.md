# Timer

Timer is a productivity mini project for countdown workflows.

## Features

- Preset countdown durations.
- Custom minutes and seconds form.
- Start, pause, resume, reset, and restart actions.
- Completed state with session count.
- LocalStorage persistence for the selected duration.

## Angular Coverage

- Standalone component.
- Signals and computed timer state.
- Interval cleanup with `DestroyRef`.
- Template-driven form bindings.
- LocalStorage persistence.
- Focused unit tests and Playwright workflow coverage.
