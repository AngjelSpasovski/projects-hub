# Calculator

## Goal

Create a compact calculator mini project that demonstrates Angular component state, keyboard input, validation, and responsive SCSS.

## Features

- Basic arithmetic: add, subtract, multiply, divide
- Decimal input
- Percentage conversion
- Sign toggle
- Clear/reset
- Keyboard support
- Error state for invalid operations such as division by zero

## Angular Concepts

- Standalone component
- Host listener for keyboard events
- Component-local state
- Template-driven button rendering
- Unit tests for calculator behavior

## Known Limitations

- The calculator evaluates one operation at a time.
- It does not yet support parentheses or operation precedence chains.
