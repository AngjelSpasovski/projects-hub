# To-Do List

To-Do List is a compact productivity mini project for task CRUD, validation, filtering, priority labels, and LocalStorage persistence.

## Features

- Add tasks with required title validation
- Optional notes, priority, and due date fields
- Edit existing tasks without losing completed state
- Mark tasks active or completed
- Filter by all, active, and completed
- Clear completed tasks
- Reset the demo dataset
- Persist task state in `localStorage`

## Angular Coverage

- Standalone component
- Template-driven forms
- Signals and computed state
- LocalStorage persistence boundary
- Translated visible labels
- Focused unit tests for task behavior
