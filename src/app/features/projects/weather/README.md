# Weather App

## Goal

Create an API-style weather dashboard that demonstrates loading, filtering, empty, and error states without requiring an external API key.

## Features

- Simulated weather data fetch
- Loading skeletons
- Condition filter
- Average temperature summary
- Last updated timestamp
- Manual error state
- Retry action

## Angular Concepts

- Standalone component
- Signals and computed state
- Template-driven form control
- Conditional rendering for async-like states
- Unit tests with fake timers

## Known Limitations

- Data is mocked locally until a public weather API and API-key strategy are chosen.
- Forecast details are limited to current city conditions.
