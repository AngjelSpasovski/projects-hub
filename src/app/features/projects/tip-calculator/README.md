# Tip Calculator

Tip Calculator is a compact utility mini project for calculating a tip, bill total, and per-person split.

## Features

- Bill amount input with decimal sanitization.
- Tip presets plus a custom tip percentage field.
- People count validation for split totals.
- Tip total, bill per person, tip per person, and total per person.
- Reset action that restores the demo values.

## Angular Coverage

- Standalone component.
- Signals and computed values for all derived totals.
- Template validation states.
- Theme-aware SCSS layout aligned with the recent productivity projects.
- Unit tests for calculations, validation, sanitization, and reset behavior.
