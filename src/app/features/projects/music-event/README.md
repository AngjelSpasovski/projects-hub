# Music Event App

## Goal

Create an event catalog that demonstrates filtering, PrimeNG controls, and modal detail views inside the portfolio workspace.

## Features

- Event category filter
- PrimeNG MultiSelect tag filter
- Result and seat summaries
- Detailed list layout
- PrimeNG Dialog event details
- Empty state and filter reset

## Angular Concepts

- Standalone component
- Signals and computed state
- Template-driven forms
- PrimeNG MultiSelect and Dialog
- Unit tests for filtering and modal state

## Known Limitations

- Event data is local mock data.
- Booking and checkout flows are intentionally not implemented yet.
