# Tic Tac Toe

## Goal

Create a small game project that demonstrates Angular signals, derived state, conditional templates, and simple game flow.

## Features

- Two-player local turns
- Winner detection for rows, columns, and diagonals
- Draw detection
- Winning-cell highlight
- Scoreboard for X wins, O wins, and draws
- Confirmation modals for restarting active rounds and resetting match score
- Round result and match winner modals
- Separate restart-game and reset-score actions
- Responsive layout

## Angular Concepts

- Standalone component
- Signals and computed state
- Template control flow
- PrimeNG Dialog
- Button grid rendering
- Unit tests for game behavior

## Known Limitations

- The game is local only and does not include AI.
- Player names and score history are not persisted yet.
