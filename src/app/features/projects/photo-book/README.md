# Photo Book

Photo Book is a visual gallery mini project for the Projects Hub portfolio.

## Features

- Grid and list gallery views.
- Search by title, location, category, or description.
- Category filtering.
- Selected photo detail panel.
- Previous and next slider controls.
- Start/stop automatic slideshow.
- Left and right keyboard navigation.
- Empty state with reset action.

## Architecture

- Standalone Angular component with signal-based state.
- Local demo photo data with optimized assets and Wikimedia Commons source links.
- All visible labels are translated through `en.json` and `mk.json`.
