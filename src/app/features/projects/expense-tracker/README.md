# Expense Tracker

Expense Tracker is a compact finance mini project for local income and expense tracking.

## Features

- Add income and expense entries
- Required title and positive amount validation
- Category, date, and type fields
- Edit and delete existing entries
- Filter all, income, and expense records
- Calculate income, expenses, and balance
- Render a Chart.js doughnut summary
- Persist entries in `localStorage`

## Angular Coverage

- Standalone component
- Template-driven forms
- Signals and computed totals
- Chart.js canvas integration
- LocalStorage persistence boundary
- Translated visible labels
- Focused unit tests for finance behavior
