# Digital Clock

Digital Clock is a live time utility mini project.

## Features

- Live ticking digital time.
- Full date display.
- 12-hour and 24-hour modes.
- Timezone selection.
- LocalStorage persistence for clock settings.

## Angular Coverage

- Standalone component.
- Signals and computed clock labels.
- Interval cleanup with `DestroyRef`.
- Template-driven settings controls.
- `Intl.DateTimeFormat` timezone formatting.
- Focused unit tests and Playwright workflow coverage.
