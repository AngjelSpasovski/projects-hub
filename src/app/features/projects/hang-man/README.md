# Hang Man

## Goal

Create a compact word-guessing game that demonstrates derived Angular state, keyboard input, validation, and responsive game UI.

## Features

- Hidden word display
- Letter keyboard
- Physical keyboard support
- Wrong guess tracking
- Remaining attempts meter
- Win and loss states
- Reset action with a new word

## Angular Concepts

- Standalone component
- Signals and computed state
- Host listener for keyboard input
- Template control flow
- Unit tests for game behavior

## Known Limitations

- The word list is local and intentionally small.
- Score history is not persisted yet.
