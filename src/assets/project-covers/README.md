# Project Cover Assets

Catalog covers use a consistent illustrated system:

- Card frame: `16:10` aspect ratio.
- Source asset: landscape PNG, at least `1536 x 960`; keep critical content inside the centered `16:10` safe area.
- Style: geometric editorial illustration with strong silhouettes and restrained accents.
- Palette: charcoal, white, pale blue-gray, cobalt, plus one or two project-specific accents.
- Avoid text, logos, watermarks, browser chrome, UI screenshots, and decorative card frames inside the image.

Live application screenshots belong in `src/assets/project-screenshots/` and may be used by project READMEs or a future gallery. Running `npm run covers` refreshes those screenshots and must not overwrite illustrated catalog covers.
