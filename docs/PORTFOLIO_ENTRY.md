## Projects Hub

An Angular and TypeScript portfolio workspace that unifies small frontend projects behind a responsive admin dashboard. It demonstrates standalone components, signals, lazy routes, reusable UI, themes, Macedonian and English localization, PrimeNG controls, validation, automated tests, and GitHub Pages delivery.

- Repository: https://github.com/AngjelSpasovski/projects-hub
- Live application: https://angjelspasovski.github.io/projects-hub/
- Stack: Angular, TypeScript, SCSS, Bootstrap, PrimeNG, ngx-translate, Jasmine, Playwright

Use this entry in the external portfolio project list. The repository URL is the canonical code-review link.
